# 🌲 Forest Guardian AI - Forest Fire Monitoring System (C++)

Forest Guardian AI is a C++ based forest fire monitoring system that simulates sensor networks, fire risk assessment, emergency alerts, evacuation management, and incident logging. An interactive web dashboard was developed using HTML, CSS, JavaScript, and Chart.js to visualize sensor readings, fire risk analysis, zone monitoring, AI recommendations, drone surveillance, and incident reports.

## Technologies Used

- C++
- HTML
- CSS
- JavaScript
- Chart.js

## Overview

Forest Guardian AI is a smart forest fire monitoring system developed using C++ for risk analysis and decision-making logic. The system processes environmental parameters such as temperature, humidity, and smoke levels to detect potential fire hazards and provide monitoring support through an interactive dashboard.

The project helps identify high-risk forest zones, generate alerts, track incidents, and visualize environmental conditions for better forest management.
